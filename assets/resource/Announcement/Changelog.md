## ⚙ **GUI 功能更新**（**需要安装新运行库**）

本次发布的版本及以后的新版本都将需要 **.NET 10** 运行库（之前是 .NET 8）才能正常运行。

请在启动 MaaYuan 前先找到并**双击**文件夹内的 （Win 系统）`DependencySetup_依赖库安装_win.bat` 或 （Mac 系统） `DependencySetup_依赖库安装_mac.sh` ，尝试自动安装新的运行库。安装完成后可能会提醒**需要重启电脑**，请务必重启。

如自动安装过程中出现问题，请访问[官方下载页面](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0)，并在页面中找到`.NET 桌面运行时 10.0.0 - Windows - 安装程序 - x64`或 `.NET 运行时 10.0.0 - MacOS - 安装程序`（M 芯片选择 `Arm64` 版、Intel 芯片选择 `x64` 版）。手动下载并双击**安装**，安装完成后可能会提醒**需要重启电脑**，请务必重启。

- 任务列表右上角新增按钮【重置】，一键将列表恢复成初始设置（日常一条龙模式）
- 右键点击任务，新增功能【从此项开始运行（后续所有已勾选的任务）】
- 界面美化调整，并新增功能【设置背景图片】、【字体大小缩放】
